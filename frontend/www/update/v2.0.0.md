# v2.0.0 업데이트 내용 (대규모 방치형 RPG 전환)

안녕하세요, 용사님들! 🎉
수작업에 의존하던 전통적인 턴제 전투 방식이 **방치형(Idle) 자동 전투 시스템**으로 전면 개편되었습니다. 주요 변경 사항은 아래와 같습니다:

---

### ⚔️ 자동 전투(Idle) 시스템 전면 도입
- 번거롭게 몬스터를 누르고 스킬을 선택하던 **수동 타겟팅 및 스킬 UI가 완전히 자동화**되었습니다.
- 이제 용사는 보유한 스탯과 몬스터 숫자에 맞춰 상황에 맞는 스킬(강 공격, 휩쓸기 등)을 스스로 골라 사용합니다.
- 한 층의 몬스터를 전멸시키면 일일이 결과창을 누를 필요 없이 **즉시 다음 층으로 끊임없이 돌격**합니다!

### 💾 아이템 및 게임오버 저장소 안정화 (자동 저장)
- 게임을 종료하거나, 체력이 다해 패배해도 모든 재화(골드), 인벤토리 아이템, 능력치를 온전히 보존합니다!
- 패배 시 1층에서부터 다시 자동으로 등반을 시작하며(환생), 브라우저를 재접속할 경우 마지막으로 도달했던 **가장 깊은 층수, 획득 기록을 한눈에 볼 수 있는 오프라인 기록 모달**이 열립니다.

### ⛺ 아이템 판매 상점 시스템 도입 및 상시 모달
- 마을이나 특정 시점에만 방문하던 상점을 **언제 어디서든** 하단 [⛺ 상점] 버튼을 눌러 부를 수 있습니다.
- 전리품이나 잉여 장비들이 인벤토리 공간만 차지하는 것을 막기 위해 **아이템 [판매] 버튼**이 직관적으로 부활했습니다! 구매가의 50% 골드를 돌려받으세요.
- 더불어 몬스터와 피 튀기는 전투가 벌어지는 와중에도 상점, 방어구 교체, 물약, 스탯 화면을 자유롭게 열고 기능을 사용할 수 있습니다!

### 🛠️ 기타 편의성 및 버그 개선
- 자동 사냥을 언제든 [⏸️ 자동 정지] 할 수 있는 토글 버튼이 생겼습니다.
- 자동 턴이 지나갈 때 전투 물약을 사용해도 불필요한 알림 모달이 뜨지 않도록 UI 템포를 조절했습니다.
- 게임 중 [🏠 홈으로] 버튼을 눌렀을 때 간헐적으로 알림창이 증발하던 네이티브 브라우저 이슈를 수정하고, **전용 맞춤형 HTML 커스텀 모달**로 안전하게 교체했습니다.
